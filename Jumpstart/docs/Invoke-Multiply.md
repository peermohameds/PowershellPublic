# Invoke-Multiply

## SYNOPSIS
 Multiplies given 2 numbers 

## SYNTAX

```
Invoke-Multiply [[-A] <Int32>] [[-B] <Int32>] [<CommonParameters>]
```

## DESCRIPTION
 Multiplies given 2 numbers and gives the results. 

## EXAMPLES

### Example 1
```powershell
PS C:\>  invoke-multiply -A 10 -B 20 
200
```

 multiplies 10 by 20 and results 200
